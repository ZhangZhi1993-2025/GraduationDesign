 Meetup

所有创客空间会员和硬件爱好者们线下交流的活动。展示项目、寻求合作、招募人才

 无需报名

时间：每周四晚7:30

Meetup是创客空间会员们和爱好者们线下交流互动的活动。每人有3-5分钟时间介绍自己的项目或者自己想做的内容。可以通过交流讨论组团队完成项目。所有人都可以免费参加。

时间：双周，每周三晚7:30

C2D2分享会主题涵盖艺术、设计、科技的各个方面，包含了众多有趣创客项目和软硬件结合创业者。促进跨界合作和产品展示交流。所有人都可以免费参加。

时间：每周五、六

工作坊为大家提供了一个动手实践的平台。邀请创客前辈们带领大家制作各种各样的作品。例如：Arduino开发，音乐播放器，3D打印，图形化编程，手工DIY，激光雕刻，激光键盘制作等。

时间：不定期

与合作公司共同主办一些活动，以往合作过的有：法国大使馆、联想、奥迪、Intel、CSDN、Google开发者社区等。

时间：每次不定期

从2012年开始举办Hackathon创想马拉松活动，集合艺术家、设计师、工程师们聚在一起，48小时内从不认识到组队做项目，最后每组队伍做项目陈述与展示。内容包括只能硬件开发、工业设计、软件开发、视频短片等部分。合作过的公司有：Intel，联想等。先后举办过：Hack for Air,Hack For Kids,Hack for Rural Dream等创想48小时活动。

时间：每次不定期

Creatica创意教育希望成为现有教育体系的补充，引导孩子们打破固有观念和思路，动手、思考、争论、创造。引导孩子们成为具有发明、设计、规划、决策能力的创造性人才。我们将邀请创意领域的佼佼者和先行者和孩子们分享、共同完成一些作品。

时间：每年举办

创客嘉年华始于2012年，由北京创客空间主办发起，随后同上海DFRobrt和新车间共同打造中国最大的创客活动品牌。目前已经在北京、上海举办过4届。吸引超过20个国家，数百位创客参加。是一个两天的创客聚会，全球创客展示项目、体验科技、设计和艺术的视觉盛宴。

时间：每月不定期举办

智造工作坊是系列工具学习和体验工作坊。包括：激光切割机、3D打印机、CNC数控机床、车床、铣床等工作坊。教大家使用工具设备做出自己的作品。
