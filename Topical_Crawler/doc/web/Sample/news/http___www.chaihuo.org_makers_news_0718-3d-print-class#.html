柴火活动日记：7月18日创客3D打印体验课堂

xhbang Posted on 2015/07/20 06:20:24 AT 创客文化
标签 无

这周六，易生3D打印在柴火举办创客3D打印体验课堂。

因为名额只有十个人，在活动开展前，还有很多人背着电脑来问：“我带着电脑，可以参加吗？”小编也只能含泪拒绝。大家想要参加柴火的活动，还是要提前报名哦。

这次工作坊主要是从3方面教大家认识3D打印机。

首先，简单向大家介绍3D打印的原理，3D材料分类和以及不同材料，不同打印机需要采取不同的打印的模式，包括PLA线条，以及EPC线条的特点以及一些应用。


大家都很踊跃了解3D打印机的运作模式。


 


本次活动最小的只有11岁的参与者，自己带着电脑来参加我们的活动，并认真地听讲和大哥哥讨论课程内容。可见科技的参与度越来越低龄化。


第二部分，老师讲述了3D打印材料的匹配度，材料和机器匹配。


最后，在导师的指导下，大家尝试使用3D软件cura，自己操作切片，尝试在操作事先建好模型的绿巨人。

 

因为打印一个绿巨人需要7个小时，在本次活动没法完整打印一个绿巨人，但贴心的易生事先打印了几个，所以还是有展示环节的，以及不能少的大合照的。


 

 


不懂3D打印的小编报道完毕~
