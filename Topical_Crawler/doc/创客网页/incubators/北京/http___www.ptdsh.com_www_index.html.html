普天德胜科技孵化器简介
北京普天德胜科技孵化器有限公司（以下简称“普天德胜”）位于北京市西城区新街口外大街28号，成立于2002年11月，是依托国务院国资委下属的中国普天信息产业股份有限公司的产业背景和中关村自主创新示范区德胜科技园区政策、地域背景成立的定位于信息通信技术领域的科技企业孵化器，并被认定为国家级科技企业孵化器、北京市战略性新兴产业孵育基地、北京市高新技术产业专业孵化基地、北京市国际科技合作基地、西城区自主创新示范基地等。
