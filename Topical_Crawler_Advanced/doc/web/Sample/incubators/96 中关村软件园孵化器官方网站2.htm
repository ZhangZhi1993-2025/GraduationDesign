介绍

【大】 【中】 【小】

中关村软件园已经与16个国家和地区的相关机构建立了业务联系，与10个国家科技园建立了友好园区关系,每年举办数十场各种形式的技术研讨会、商务洽谈会、品牌推广会，组织园内外企业参加美国、欧洲、日本、印度以及国内有关展会，共同拓展国内外市场。

2012年园区共有来自8个国家的30家国际企业，出口业务规模10.7亿美元（68亿元人民币），海外分支机构数量达177家，创立TD-SCDMA、McWiLL宽带多媒体集群系统2项国际标准，17家企业参与国际标准制定，国际专利189项，国际注册商标30项，外籍员工达2733人，广联达、华胜天成、启明星辰、博彦科技、文思创新、网秦天下等入园企业开展多起国际并购，与印度TCS的合资公司、中芬金桥创新中心、中以国际技术创新合作转移中心等均已落户园区，成功设立“中关村软件园芬兰创新技术与转移中心”，园区已成为展示我国高科技产业发展成果、国际合作和技术转移的重要窗口。

2012年12月,中关村软件园孵化器承接市科委科技服务业促进项目，在鼎好大厦建设“软件与信息领域国际技术转移”项目，旨在与国际知名孵化机构、科研院所、企业等开展合作，致力于将国外先进技术和最新研发成果引入园区，同时将园区企业技术需求与技术供给形成有效对接，利用双方资源的优势互补及创新的机制和商业模式，搭建高效的国际技术转移和成果转化产业化平台。

平台服务内容：

n  提供技术项目“软着陆”服务，包括工商注册、行政许可、财务咨询、法律、知识产权、融资咨询等，以及商务助理、办公场所、集中办公区等服务。

n  了解国际技术转移供给方及国内技术转移需求方的需求；

n  做引进技术的市场评估和可行性分析；

n  信息服务，包括潜在客户联系、商业信息等；

n  展示推介及对接服务，包括技术项目展示推介、需求挖掘与项目对接；
