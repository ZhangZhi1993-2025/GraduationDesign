园区介绍

北京北达燕园科技孵化器有限公司成立于2002年5月, 由北京北大科技园有限公司控股,注册资本5000万元人民币.是北京大学和中关村科技园区管委会共建的创业孵化器基地；是北京市科委认定的北京市高新技术企业孵化基地；是北京市人事局认定的北京市留学人员创业园。本公司集孵化、投资和运营顾问服务三位一体。是专门吸引海外留学人员回国创办高科技企业，为具有原创自主知识产权的项目提供全程创业孵化及促进科技成果转化的创业服务机构。

创业服务体系：

创业行政服务、投融资服务、培训服务、企业促进服务、物业服务、其他公共商务服务。

北大孵化器广泛吸收北京大学及周边的创新技术、管理思想和人文精神，致力于成为中国最佳的企业创新中心。

孵化模式

咨询辅导、融资、营销+技术转化=成功创业

咨询辅导：（MENTOR）提供创业服务：提供商务服务：提供生活服务：提供物业服务。

融资：（MONEY）北大留学人员创业园对优秀的入园企业提供“种子”资金：帮助引进外部投资和风险投资：为园项目羡慕提供融资担保和服务，切实解决创业者的资金困难。

营销渠道：（MARKETING）北大留学人员创业园为入园企业提供企业发展战略、市场营销、与公司治理结构结构方面的顾问，减少失误，走向成功。

技术成果转化：（TLO）北大留学人员创业园通过学校资料和科研院所的技术转移中心帮助企业成果转化和引进。
