    全球最大的众筹平台之一 Indiegogo 来中国啦！


    Indiegogo 作为全球最大的众筹平台之一，2014年的月访问量已超过1340万。并且成功帮助亿航、Crazybaby、壹枱钢琴、阿尔法机器人等众多国内厂商进行海外市场推广。


    12月，Indiegogo 硬件设计类总监 Ben Bateman 将带来“China Pilot Program”计划，旨在发现更多优秀的国内早期智能硬件创业公司，并帮助他们进行海外市场推广。北京创客空间联合创始人 Joanna Wei 正式被 Indiegogo 官方聘请为在华布道师，将全面参与到 China Pilot Program 中。


    创客空间孵化器 Makebator 作为 Indiegogo 中国区合作伙伴，将共同扶持中国智能硬件创业团队进行海外众筹、市场拓展等工作。


    中国众筹市场和美国众筹市场有哪些差异？

    中国产品做众筹时通常会遇到的困难？

    对中国团队拓展美国市场有什么更好的帮助？


    想了解更多详情，快来参加12月10号创客空间孵化器和 Indiegogo 联合主办的分享活动吧。

    Indiegogo 介绍：

    Indiegogo 于2008年成立，总部坐落于旧金山。在洛杉矶、纽约都设有分部。作为全球最大的众筹平台，每周都有来自全球的创客都会通过Indiegogo 筹集超过百万美元的资金来启动创业项目或者尝试新的商业点子。这是一个面向所有科技，硬件，以及任何创意项目的众筹平台。


    ChinaSF 介绍：

    旧金山市驻华办公室是旧金山和中国之间的桥梁。自2008年以来，旧金山市驻华办公室带来了50多家公司，并通过支持中国企业和投资者创造了旧金山湾区的350多个就业岗位。通过其在北京，上海和旧金山设有办事处，旧金山市驻华办公室指导湾区企业寻求扩大到大中国区。旧金山市驻华办公室是市长办公室和旧金山经济发展中心的合资企业。


    创客孵化器介绍：

    创客孵化器是北京创客空间在创客3.0的理念下，打造的硬件孵化器2.0模式，致力于通过深度孵化和全方位产业结合的模式来提升创客、智能硬件创业项目的发展模式，更新硬件创业迭代逻辑，融合上下游产业背景资源，让创客、智能硬件创业公司能够获得充分指导、渠道、销售、产业背景的资源。
