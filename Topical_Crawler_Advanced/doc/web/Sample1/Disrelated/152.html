南方周末新媒体•2010暑期实习计划

infzm 发布于 2010-07-07 14:30:52

南方周末新媒体事业部提供以下实习岗位，欢迎对具有新闻理想和南周情结，认同新媒体的未来趋势，对互联网和移动终端感兴趣的同学加入。

市场助理（2人）

职责：

1. 协助市场经理进行南周新媒体的品牌维护、市场策划工作。

2. 协助市场经理对南周商城做出规划，保证商城正常运转，并担负制定商城促销方案、更新商品信息、统计销售数据、商品配送等工作。

3. 撰写市场宣传文案和活动策划方案，并负责市场活动的具体执行环节。

4. 与南周新媒体乃至整个南周报系的客户、合作伙伴进行沟通，维护良好的客户关系。

岗位要求：

1. 具有良好的文字驾驭能力和语言表达能力，能够独立思考，具有良好的逻辑思维能力。

2. 热心公益，最后具备一定的公益/市场活动策划和执行经验。

3. 善于学习，有责任心和团队合作意识，具备优秀的沟通和表达能力。

4. 能够独立跟进项目的运作，可以适应短期出差的需要。

5. 实习时间在3个月以上，优先考虑大三学生。

工作地点：广州

 

视觉助理（1人）

职责：

1. 协助设计师改进网站产品UI及视觉效果。

2. 协助图片编辑发布图片新闻。

岗位要求：

1. 能熟练使用Photoshop、AI等设计软件，并具备基本的设计表达和构图能力。

2. 对新闻摄影有一定了解

3. 认真负责，思维清晰，有一定审美能力

4. 实习时间在3个月以上，优先考虑大三学生

工作地点：广州

 

联系方式

请将您的简历发至 new_media@infzm.com 并在邮件名注明“XXX应聘XXX实习岗位”，合则约见。
