香港电子展：创新功力还需历练的“创客空间”

2015-10-19 20:01 原创 赵悟空
1条评论

智能硬件的大热，让很多爱好者以及初创公司也对智能硬件的研发与制作产生了兴趣，从而在智能硬件领域涌入了一群创新者，这群创新者通常就被人们称为——“创客”。

“创客”这一名词越来越流行，业界也对“创客”群体越来越重视。此次在香港“环球资源移动电子展”（Global Sources Mobile Electronics）上，还专门为创客们开辟了一个展览空间——“创客空间”专区，雷锋网记者在现场还真看到了几个有意思的东西。

LED 环保灯

在当代的社会，环保越来越受人重视，由此也涌现了一大批环保主题的科技产品，这款 LED 环保灯就是其中之一，这个奇形怪状的灯泡是由一片片带有LED灯的电路板组成，外观比较非常的个性。

在现场，这款环保灯的开发者为雷锋网记者演示了这款LED照明灯与普通白炽灯照明时的功率对比，这两款灯在亮度差不多的时候，环保灯的功率消耗明显大幅度小于白炽灯。如此看来，这款灯的环保节能效果真是非常不错，不过一般这种高科技环保产品都价格不菲，如果不控制好价格，那么环保产品想在市场上普及还有一定的难度。

自发电门铃

一般现阶段家里的门铃都采用走线的方式将按钮和扬声器连接在一起，通常情况下，还需要连接电源，并且为了美观还将电线尽量隐藏，这样就加大了修理的难度，而在“创客空间”展示的这款自发电门铃就解决了这两个问题。

这个门铃带有自发电无线模块，按动开关另一边的门铃就会响，这其中完全靠无线连接，并自行发电无需外接电源，算是一款贴心的产品。

智能镜子

在我们预想中的智能家庭中，一个智能镜子似乎也是构建智能生活的必备之一，在“创客空间”展区也有这么一块智能镜子。

这块镜子在底部内置了一块显示屏，采用了Android开发的智能系统，可以实现观看视频和天气预报等功能。不过在现场看，效果并不算很好，另外，目前人们对智能镜子的并没有真正的需求，花高成本买一块内置显示屏的镜子并不划算，所以智能镜子向推向市场还需要好好考虑一下客户需求。

空调控制器

目前市面上的智能家居产品中，智能空调、遥控器之类的产品算是我们接触频率最高的了，而雷锋网记者在展会上看到的这款控制器还显得比较特殊。

这款控制器可以控制所有红外控制的空调，有手机用的配套APP，其亮点在于，连接WiFi之后，控制器可以接收外面的天气情况以及其他环境和客户个人数据，并基于诸多数据全自动为客户调节事宜的温度。概念的非常的不错，没有经过测试，不知实际使用效果如何。

总结

其实“创客空间”这个展区并没有预想中带给我们很多惊喜，包括模块化手机都没能做到让人亮眼。可能因为其“为初创企业而设”的定位有关，其中展示的产品多是一些带有商业性质并且创新程度并不是很强的一些成品。科技的发展需要创新，智能硬件时代也需要创客们的创新来解决一些生活实用的问题。不过目前很多创客多看不懂客户的真正需求，所以导致研发的产品定位上有所偏颇，智能硬件是个新兴的领域，这个领域的发展还需要创客们的不断摸索才能实现。
