中关村科技园区西城园管理委员会关于落实《北京市西城区支持中关村科技园区西城园自主创新若干规定》修订版的实施办法

发布单位：

文　　号：


   根据国务院及北京市《关于税收等优惠政策相关事项的通知》精神，区政府对原《北京市西城区支持中关村科技园区德胜科技园自主创新若干规定》部分条款进行了修订调整，并经2015年第111次区政府专题会审议通过。为落实新修订的《北京市西城区支持中关村科技园区西城园自主创新若干规定》（西政发〔2012〕14号，2015年8月修订）（以下简称《若干规定》），中关村科技园区西城园管理委员会（以下简称“西城园管委会”）特制定本实施办法。

一、关于《若干规定》第四条、第五条（一）、第六条、第十六条等有关项目配套资金的落实

（一）支持范围

科技研发项目

1. 国家级项目：主要指科学技术部的“863”计划、“973”计划、科技支撑计划、中小企业创新基金、重点新产品、“火炬”计划、重大新药创制（含卫生部批复项目）、重大科技专项、重点科技产业化、科技成果转化项目；工业和信息化部的电子信息产业发展基金项目；工业和信息化部、财政部等共同实施的物联网、云计算等新兴产业发展专项资金项目。

2. 北京市级项目：主要指中关村中小企业技术创新专项、公共专业技术支撑服务平台专项、产业发展资金项目、现代服务业试点项目；北京市科学技术委员会中小企业技术创新专项、科技计划项目、科技研究开发机构自主创新专项、设计提升计划项目、科技成果转化项目、重大科技专项、科技服务业专项、创新环境与服务体系建设专项。

企业自建研发机构

是指企业作为申报和实施主体，获得国家科技部、国家工业和信息化部、国家发改委、市科委、市发改委、市经信委、中关村科技园区立项并给予专项资金支持的研发中心、技术中心、工程中心、检测中心、实验室、设计创新中心等企业研发机构建设项目。

市重大产业化项目

是指列为市级（含中关村）重大产业化项目并获得专项支持资金。

科技中介服务专项

园区科技中介企业、西城区高新技术产业孵化基地（含自主创新示范基地）获得的国家、本市和中关村相关科技中介服务专项资金。

（二）申报条件

1. 企业在西城区注册纳税，并且是项目的申报和实施主体；参加产业技术联盟的企业需是经过北京市相关部门认定的产业联盟的发起单位或联盟中排名前5位的企业。

2. 企业申报科技研发项目配套资金的，其申报的项目不属于固定资产投资类、改造类、配套类、交流合作类项目。

3. 企业获得的科技研发项目配套资金可核定以下间接费用：承担单位为项目研发提供的现有仪器设备和房屋及日常水、电、气、暖消耗支出的科技条件支撑费，协调管理费，监督检查费，科研人员激励费（不超过配套资金5%比例）。

4. 西城园管委会按照北京市重大产业化项目无偿资助和股权投资两种方式，给予相应的配套资金。其中股权投资方式的配套资金按照市级专项资金入股企业的比例对应入股投资。

5. 企业同一项目分别获得国家和北京市级项目资金支持的，按项目获得的高限资金支持额度配套。

（三）申报材料

1. 项目配套资金书面申请及汇总表；

2. 企业营业执照副本复印件、税务登记证副本复印件；

3. 中关村高新技术企业证书复印件（区级科技孵化机构不需提供该项材料）；

4. 获得国家、北京市、中关村专项的相关证明材料及资金到位证明；

5. 企业上年度财务审计报告；

6. 企业申报材料真实性承诺函。

以申报上材料须一式两份，并加盖企业公章。

二、关于《若干规定》第五条（二）联合实验室支持条款的落实

（一）支持条件

1. 联合实验室须是企业与高等院校、科研院所合作建立，且企业与高等院校、科研院所没有出资或隶属关系；

2. 联合实验室建成投入使用满一年，已取得完整的实验成果，专项核算。

（二）申报材料

1. 联合实验室支持书面申请；

2. 企业营业执照副本复印件、税务登记证副本复印件；

3. 中关村高新技术企业证书复印件；

4. 共建联合实验室的协议；

5. 实验室人员、设备、场地清单及有关照片；

6. 实验室已取得的实验成果报告及有关证明材料。

三、关于《若干规定》第七条、第八条有关企业专项奖励的落实

（一）奖励标准与范围

1. 国家高新技术企业首次认定奖励：

由西城园管委会受理首次申报通过国家高新技术企业认定，并且当年度或之前年度已认定为中关村高新技术企业的企业，给予一次性5万元奖励。

2. 企业获奖奖励：

（1）企业年度获得以下奖项之一的，给予一次性奖励10万元：国家科学技术进步奖二等奖及以上奖项；中国专利奖金奖、优秀奖；中宣部精神文明建设“五个一工程”；德国红点奖（红点、红点之星、红点至尊三个级别奖项）；德国IF奖（金奖、IF奖）；美国IDEA奖（银奖及以上奖项）；美国电影声音剪辑协会专业技术金卷轴奖；美国奥斯卡金像奖中的最佳音效、音响效果、视觉效果、最佳剪辑、服装设计等技术性奖项；文化部原创动漫扶持计划；

（2）企业年度获得以下奖项之一的，给予一次性奖励8万元：中国创新设计“红星奖”（银奖及以上奖项）；中国优秀工业设计奖；中国文化艺术政府奖动漫奖；世界经济论坛“科技先锋”；国家规划布局内重点软件企业；中国互联网企业100强；中国香港金像奖或中国台湾金马奖中设置的最佳音效、音响效果、视觉效果、最佳剪辑、服装设计等技术性奖项。

（3）企业年度获得以下奖项之一的，给予一次性奖励5万元：北京市科学技术奖二等奖及以上奖项；北京市发明专利奖特等奖、一等奖；中国网络电视台“中国年度品牌”；德勤“高科技高成长中国50强”。

（二）奖励要求

1. 国家高新技术企业认定单独奖励。

2. 企业的同一产品（技术、项目等）获得多个奖项的，按照其中一个最高奖励额给予一次性奖励。

3. 每年度，企业的多项产品（技术、项目等）获得多个国际、国家、市级奖项的，按照国际和国家奖项中一项最高奖励额奖项、北京市级一项奖项分别给予奖励。

4. 企业需将当年度获得的全部奖项填报完整，少报、漏报，一经发现即取消对该企业的获奖奖励资金。

（三）申报材料

1. 书面申请及企业年度获得全部奖项的汇总表；

2. 企业营业执照副本复印件、税务登记证副本复印件；

3. 中关村高新技术企业证书复印件；

4. 专项证明材料：

企业申报国家高新技术企业认定奖励还须提交国家高新技术企业证书复印件；

企业申报获奖奖励还须提交获奖证书、通知等相关证明材料。

四、关于《若干规定》第九条、第十条有关知识产权和技术标准条款的落实

（一）申报条件

知识产权

1. 企业申请实用新型、外观设计、软件著作权、注册商标、集成电路布图、植物新品种6种知识产权，需每年作为第一申请人实际原始取得的以上知识产权满3项方可享受补贴。

2. 企业作为第一申请人实际获得授权的发明专利按照当年内实际授权数补贴，年度补贴总额不设上限。

技术标准

具体细则说明将与有关部门另行公布。在有关细则未公布前暂时参照不超过原《德胜科技园管委会关于落实知识产权和技术标准支持的实施办法》（德胜管发〔2012〕9号）标准进行政策申报。

（二）申报材料

1. 知识产权、技术标准书面申请；

2. 企业营业执照副本复印件、税务登记证副本复印件；

3. 中关村高新技术企业证书复印件；

4. 已获得知识产权，或已公布技术标准，或已设立标准化技术委员会、分技术委员会、工作组秘书处等相关证明材料；

5. 申请知识产权补贴的须提供知识产权申报汇总表，申请技术标准补贴的须按照西城区质量技术监督局要求填报并提交有关技术标准补贴材料。

五、关于《若干规定》第十一条企业贷款贴息条款的落实

（一）支持范围

1. 中关村科技担保公司、北京首创担保公司担保的企业贷款；

2. 与西城园管委会（含原机构）签订合作协议的金融机构发放企业的贷款；

3. 企业向银行申请获得的研发项目专项贷款。

（二）申报条件

1. 企业是贷款的主体，不支持个人名义贷款。

2. 企业同笔贷款实际期限最短不少于两个月，最长不超过二十四个月。

3. 企业在申报前已完成贷款的还本付息。对于在申报年6月以前未到期贷款，归入到下一年度贴息支持。

4. 因企业自身因素导致超期还贷的，不予贴息；企业贷款发生代偿的，三年内不再享受贷款贴息支持。

（三）申报材料

1. 书面申请及贷款贴息申报汇总表；

2. 企业营业执照副本复印件、税务登记证副本复印件；

3. 中关村高新技术企业证书复印件；

4. 企业与银行签订的贷款合同复印件，担保贷款的须附担保合同复印件；

5. 企业还本付息的相关票据复印件。

六、关于《若干规定》第十二条投融资风险补贴条款的落实

（一）支持范围

1. 在西城区注册纳税并与西城园管委会（含原机构）签订合作协议的金融机构（以下简称“合作金融机构”）；

2. 在西城区注册纳税的各类投资机构；

3. 西城区高新技术产业孵化基地（含自主创新示范基地）（以下简称区级科技孵化机构）。

（二）申报条件

合作金融机构

1. 以优惠利率（最高利率不超过人民银行同期贷款基准利率上浮30%的标准）向园区内高新技术企业提供的贷款。

2. 风险补贴支持的贷款须是不通过担保公司担保的贷款。

3. 年度实际融资总额的计算：企业单笔贷款期限不满3个月的，贷款金额不计入年度实际融资总额；企业单笔贷款期限满3-9个月的，按照实际贷款额50%额度计入年度实际融资总额；企业单笔贷款期限在10个月至1年的，按照实际贷款全额计入年度实际融资总额；金融机构为企业开展的信用证、保函等中间业务，按照实际发生额1‰额度计入年度实际融资总额。

各类投资机构

1. 投资机构的注册资本不低于5000万元。

2. 以货币形式向园区内中关村高新技术企业投资，且同笔投资年限不低于3年。

3. 投资资金已投入被投资企业，且被投资企业已在申请年度内完成工商登记变更。

区级科技孵化机构

1. 以货币形式向入驻企业投资，进行持股孵化与培育。

2. 同笔投资年限最低不低于3年。

3. 投资资金已投入被投资企业，且被投资企业已在申请年度内完成工商登记变更。

4. 风险补贴支持的投资是基地自有资金投资，且须专项核算。

（三）申报材料

1. 投融资风险补贴书面申请；

2. 年度投融资开展情况总结；

3. 各类投融资协议、合同等复印件（金融机构贷款合同至少提供有合同号、贷款用途、贷款利率、贷款期限、贷款金额、担保方式、双方签章内容的关键页复印件）；

4. 资金到位证明（金融机构提供已支付企业贷款凭证复印件；投资机构与两类基地提供自有资金投资到位证明）；

5. 所服务企业的中关村高新技术企业证书复印件；

6. 投资机构、区级科技孵化机构还须提供年度财务审计报告。

七、关于《若干规定》第十七条有关企业中介服务费用补贴条款的落实

（一）补贴内容

1. 信用中介服务：企业信用评级、征信费用；

2. 财务中介服务：企业年度财务审计、国家高新技术企业复审需要的项目研发投入与技术性服务收入专项审计、科研项目申报必须的财务评审中介服务费用；

3. 知识产权中介服务：专利、著作权、商标代理中介服务费用及涉及知识产权保护的法律专业服务费用；

4. 并购中介服务：境内外已上市公司、场外交易市场挂牌企业、十百千工程入选企业、“瞪羚计划”重点培育企业四类企业在并购其他企业时发生的法律、财务中介服务费用；

5. 认证中介服务：企业ISO9000系列认证、ISO14000系列认证、GB/T28001认证、CMMI评估和复核、美国FDA认证、欧盟CE认证服务费用。

（二）补贴标准

1. 信用中介服务费用：每项补贴额度为实际发生额的20%，年度补贴金额不超过3000元。

2. 财务中介服务费用：每项补贴额度为实际发生额的25%，年度补贴金额不超过2万元。

3. 知识产权中介服务费用：每项补贴额度为实际发生额的20%，年度补贴金额不超过1万元。

4. 并购中介服务费用：按并购企业时实际发生的法律、财务中介服务费用20%的比例补贴，年度补贴金额不超过3万元。

5. 认证中介服务费用：

（1）ISO9000系列认证、ISO14000系列认证、GB/T28001认证服务费用，每项补贴额度为实际发生额的25%，年度单项补贴金额不超过1万元；

（2）CMMI评估和复核服务费用，三级企业年度补贴5000元，三级以上企业年度补贴1万元；

（3）美国FDA认证、欧盟CE认证，每项补贴额度为实际发生额的25%，年度补贴金额不超过3万元。

6. 同一企业以上全部费用的年度补贴总额不超过5万元。

（三）申报材料

1. 书面申请及中介服务费用分类申报汇总表；

2. 企业营业执照副本复印件、税务登记证副本复印件；

3. 中关村高新技术企业证书复印件；

4. 中关村信用促进会会员证书复印件；

5. 年度西城园协会会员缴费证明；

6. 中介服务合同或协议复印件、中介服务收费凭证（发票、银行进账单、支票票根等）复印件；

7. 根据购买中介服务类别，补充下列材料：

信用中介服务：提交信用报告、征信报告的首页复印件。

财务中介服务：提交专项审计报告首页、贴有北京注册会计师协会标志页、资产负债表、损益表、现金流量表的复印件；中介服务合同首页及签字盖章页的复印件；高新技术专项审计报告的首页、研发费用结构明细表、高新技术产品（服务）明细表、签字盖章页的复印件。

知识产权中介服务：提交有关知识产权受理通知书复印件；有关知识产权代理或法律服务合同首页及签字盖章页复印件。

并购中介服务：提交并购协议首页及签字盖章页的复印件、并购企业的股权变更证明材料。

认证中介服务：提交有关认证证书或批准文件复印件，中介服务合同复印件。

八、关于《若干规定》第十八条、第十九条、第二十条有关人才激励条款的落实

人才称号奖励

（一）申报条件

1. 获得称号的人才原则上需在园区企业服务满三年；

2. 同一人才获得多项称号，按单项最高额给予奖励；

3. 由获得称号人才所在企业申报当年度人才称号奖励。

（二）申报材料

1. 人才所在企业提交人才本人签字并加盖企业公章的书面申请；

2. 人才所在企业营业执照副本复印件、税务登记证副本复印件；

3. 人才所在企业的中关村高新技术企业证书复印件；

4. 人才本人签字的身份证复印件；

5. 获得有关人才称号的证明材料；

6. 人才获得称号当年度或次年度在北京市的个人完税证明；

7. 获得称号人才服务园区内企业的三年劳动合同复印件；对人才获得称号当年度服务企业不满三年的，需提交该人才服务企业三年的承诺书及与企业签订的不少于三年期的劳动合同复印件。

人才住房补贴

（一）申报条件

1. 企业拟订住房补贴分配方案，申报并落实住房补贴资金。

2. 确保专款专用、补贴资金分配到个人。

3. 住房补贴对象为企业高级管理人员和核心技术骨干人员。 

4. 企业年度实际补贴人数不少于2人、不高于10人。

5. 获得人才称号人员所在企业申请的住房补贴原则上不再分配给获得该项人才称号的个人。

（二）申报材料

1. 书面申请；

2. 企业营业执照副本复印件、税务登记证副本复印件；

3. 中关村高新技术企业证书复印件；

4. 企业或人才获得有关称号的证明材料；

5. 企业住房补贴分配方案、补贴人员基本情况表、补贴分配承诺书、拟补贴人员的身份证复印件。

人才医疗保障服务

（一）西城园管委会按照每名人才每年度1500元标准安排一次免费体检，体检场所定期公布。

（二）由符合条件的企业填报人才免费体检申请表，报送西城园管委会审核；每年底以前，组织核定的人才进行免费体检。

（三）每年度人才免费体检，需通过审核的人才本人参加，如本人不能在规定时间内参加，视为自动放弃，其他人员不能替代。

九、工作流程及相关要求

（一）西城园管委会每年定期发布年度政策申报通知

（二）企业在规定受理时限内向西城园管委会提交年度政策申报材料，逾期不予受理

（三）西城园管委会对提交材料齐全的，开具受理通知单

（四）西城园管委会对材料进行初审及汇总

（五）通过西城园联席会、西城园管委会主任办公会、区政府专题会等逐级审议

（六）在西城园网站上公布审议通过的企业和人才名单；审核未通过的不再另行通知

（七）按照审议结果拨付资金

十、本办法自公布之日起实施。2014年度起，《北京市西城区支持中关村科技园区西城园自主创新若干规定》的落实工作按照本办法执行。
